🎣 钓拉么 (DiaoLaMe)
智能垂钓环境预测与战术决策系统

*钓了么 是一款专为垂钓爱好者打造的高颜值、轻量级 Web 工具。系统通过整合全球实时气象数据与动态水温变化算法，将复杂的原始数据转化为直观的 0-100 钓鱼指数，并提供定制化的战术建议，让您在出发前做到“心中有数”。

✨ 核心特性
1. 核心预测算法 (Five-Axis Matrix)
系统基于五维度加权算法进行深度分析，模拟真实水域环境：

时间因素 (30%)：分析晨昏蒙影、昼夜节律。
气压稳定性 (20%)：实时追踪气压波动，判断鱼群活跃度。
天气稳定性 (20%)：过滤突发气象波动对鱼口的影响。
风况与溶氧 (15%)：分析风向风级对表层水体溶氧的贡献。
水温预测 (15%)：结合气温趋势反演水温稳定性。
2. 交互式地图定位 (Leaflet 2.0)
极致视觉：采用清爽的白色极简地图底图（Positron Style），让蓝色水域更加醒目。
沉浸式布局：地图边缘左右全宽顶满，消除容器束缚感。
智能聚焦：系统启动后自动锁定钓点周围 3 公里视野，精准寻找标点。
3. 环境预警与建议
温差红线：实时监测未来 7 天昼夜温差，当温差超过 12°C 时自动弹出红色警告，预防“断口”。
动态战术：自动区分“淡水钓”与“海钓”模式，生成对应的饵料、水层及线组建议。
🛠️ 技术底座
前端框架：Tailwind CSS (极致响应式与排版优化)

地图引擎：Leaflet.js (定制化白色滤镜)

数据接口：

Open-Meteo (气象、温差、风场数据)

Nominatim (地理编码与逆地理编码)

UI 哲学：极简主义、玻璃拟态、沉浸式卡片布局。

🚀 部署指南
快速预览
直接在浏览器中打开 index.html 即可使用，无需任何安装流程。

线上部署 (推荐)
为了确保定位精度和 API 请求的安全性，建议在 HTTPS 环境下运行：

Vercel (首选)
连接项目 GitHub 仓库。
Vercel 会自动识别并部署为静态网站。
GitHub Pages
将代码推送到仓库后，在 Settings -> Pages 中开启服务。
🎮 如何使用
锁定标点：在搜索框输入地址，或直接在地图上点击任意水域。
设置偏好：切换“淡水/海钓”模式，并调整预期的出钓时间。
智能预测：点击“开始智能预测”，系统将实时计算环境得分。
复盘战术：参考侧边栏生成的进度条拆解，阅读底部生成的专家战术提示。

本系统生成的预测分数和建议仅基于气象算法模型，属于理论参考依据。实际垂钓效果会受到当地水体能见度、电/网捕干扰、实时水流波动等不可控因素影响。请务必注意水边安全，遵守当地环保与渔业法律法规。
